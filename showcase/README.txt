LXR Mailbox — showcase folder
=============================

ui-inbox-mockup.html
  Static browser preview of inbox list styling (parchment header, gold accents, mail cards).
  Open locally: double-click or file:// — not loaded by FiveM.

tebex-feature-poster.html
  Tebex-style feature grid + hero placeholder. Export to PNG via browser full-page
  screenshot, or replace the hero block with your in-game screenshot.

visual-prompts-tebex.txt
  Copy-paste prompts for AI in-game-style art OR use as shot list when capturing real
  RedM screenshots (preferred for marketplace accuracy).

Workflow: ship static HTML previews in-repo; optional AI hero PNG (lxr-mailbox-showcase-hero-concept.png)
for layout checks — replace with real 1920×1080 RedM PNG captures per visual-prompts-tebex.txt for listings.
